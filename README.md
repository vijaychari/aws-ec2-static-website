# AWS EC2 Static Website Hosting

## Project Description
This project demonstrates hosting a static website on an AWS EC2 instance using Apache web server.

## Services Used
- AWS EC2
- Amazon Linux
- Apache (httpd)
- GitHub

## Steps Performed
1. Launched EC2 instance
2. Installed Apache
3. Uploaded static website
4. Opened HTTP port
5. Accessed website using Public IP

## Commands Used
```bash
sudo yum install httpd -y
sudo systemctl start httpd
